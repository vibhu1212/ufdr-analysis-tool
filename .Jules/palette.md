## 2024-04-05 - Streamlit Button Alignment & Tooltips
**Learning:** Using invalid kwargs like `width="stretch"` in Streamlit `st.button` fails silently, leaving buttons unaligned and poorly structured. In addition, providing tooltips (`help` parameter) is crucial for primary action buttons to improve user confidence and accessibility.
**Action:** Always use `use_container_width=True` for column-filling buttons and include `help` descriptions for UI accessibility and user guidance.

## 2024-05-03 - Streamlit Empty States for Chat
**Learning:** By default, Streamlit chat interfaces (`st.chat_message`) appear entirely blank when `st.session_state.messages` is empty, which creates a confusing "dead" UI for first-time users.
**Action:** Always implement a visually distinct "empty state" using `st.markdown(..., unsafe_allow_html=True)` with inline styles (to avoid Streamlit CSS variable issues) before rendering the main chat loop. Include helpful guidance and example queries to improve onboarding UX.
