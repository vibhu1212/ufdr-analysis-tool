## 2024-05-18 - Conversational Empty State
**Learning:** Conversational interfaces without chat history present a "blank canvas" problem that can stall user interaction. A structured empty state with clear calls to action and example queries tailored to the application's domain makes the feature instantly approachable.
**Action:** Always provide an empty state for conversational UIs when there's no history. Ensure example queries directly map to the core capabilities and value propositions of the tool (e.g. searching contacts, locations, and events in a forensic context).
